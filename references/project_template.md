# School Project Template (สรุปโครงการโรงเรียน)

## General Settings
- **Font:** TH SarabunPSK 16pt (Standard), 16pt Bold (Headers)
- **Margins:** 1 inch (1440 Twips / 914400 EMUs) all sides
- **Line Spacing:** Single (1.0), 0pt before/after

## Layout Structure
1. **Header Block:** (Project, Plan, Strategy, Standard, Type, Budget, Responsible, Duration)
2. **Separator:** Centered asterisks (********************************************)
3. **Section 1: หลักการและเหตุผล** (Indented first line)
4. **Section 2: วัตถุประสงค์** (Bullet points 2.1, 2.2, ...)
5. **Section 3: เป้าหมาย** (Sub-sections 3.1 Quantitative, 3.2 Qualitative)
6. **Section 4: กิจกรรมและขั้นตอนการดำเนินโครงการ** (4-column Table: ขั้นตอน, ระยะเวลา, ผู้รับผิดชอบ, การประเมินผล)
7. **Section 5: งบประมาณ** (6-column Table with merged header: ที่, รายการ, งบประมาณ, จำแนกตามหมวดรายจ่าย [ตอบแทน, ใช้สอย, วัสดุ])
8. **Section 6: การประเมินผล** (4-column Table: ที่, ดัชนีบ่งชี้, วิธีการ, เครื่องมือ)
9. **Section 7: ผลที่คาดว่าจะได้รับ** (Bullet points 7.1, 7.2, ...)
10. **Signatures:** Proposer, Endorser, Approver blocks with specific alignment.
